2. Phân bổ Liều tiêm

Nền (50%) 0 UI 1 lần/ngày
Bữa ăn (50% ÷ 3) 0 UI Trước 3 bữa
Sáng (2/3 TDD) 0 UI Trước ăn sáng
Chiều (1/3 TDD) 0 UI Trước ăn tối
